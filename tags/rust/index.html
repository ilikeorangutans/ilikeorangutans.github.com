Book Arrival: The Rust Programming Language

20 Nov 2018 , tagged: Rust, Books

My copy of The Rust Programming Language arrived this week and I’m excited to dig deeper into Rust. Ever since I read that Rust works without a garbage collector and yet doesn’t require the programmer to manually allocate memory my interest has been piqued. I’ve dug a little into the first chapter, and I like it so far. The syntax of the language is not exotic, it has matching which I find very nice, and the build tool seems reasonable.
